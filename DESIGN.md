## 1. Формула игры

> **Скелет Dome Keeper, начинка EBM.** (Финальный маппинг 2026-07-20, вечер.)
> Dome Keeper: купол (защищай+улучшай) + кипер (добывает ради купола и себя). Тугость — из агента, который не может быть в двух местах, и таймера волны, который гонит его домой.

| Кость Dome Keeper | EBM |
|---|---|
| Купол — стационарное сердце, его осаждают | **Город-хаб** (гномье поселение под Хартией ✅): казна, склад, переработка, найм |
| Кипер — уходит добывать, возвращается с грузом | **Башня-персонаж**: выезжает к жилам, **трюм = carry-кап**, возвращается выгрузиться |
| Кипер копает и таскает | **Рука крошит** жилы (пук-пук), **гномы пылесосят** в трюм башни |
| Апгрейды у купола между волнами | **Улучшения в хабе** (верфь-док вешает навороты ✅, кварталы = экономика, найм) + 🎯 срез-стройка нутра башни (вернулась 2026-07-21) |
| «Глубже» = богаче руда + злее волны | **Дальше от хаба** = жирнее жилы/данжи; **следующий хаб** = следующий «слой» |
| Таймер волны гонит домой | **Ночная осада на хаб** — башня должна быть дома к ночи |
| Реликвия = цель забега | Плата Вратам 900🥉 → мех + финальная осада |

**Жадность DK дословно:** «ещё одну жилу до заката» — башня далеко, трюм почти полон, солнце садится.
**Наше отличие:** кипер DK один и бессмертен; у нас башня-персонаж + **расходуемая артель** — потеря гномов это цена (не каскад, пока есть найм).

## 2. Сущности — ЧЕТЫРЕ, финал 2026-07-20

- **Башня = персонаж (кипер).** Мобильное тело игрока (WASD, дэш, таран, парри ✅ `tower.gd`), дерётся, возит груз в трюме. Навороты **вешаются** на неё в хабе (верфь-док: парковка чинит, эллинг/бронцех = кап HP, окно срезов ✅). Улучшения корпуса — ВСЕ в верфи (решение 2026-07-21), «нутро по срезам» = их визуализация в окне срезов ✅; боевой обвес = МАШИНЫ в слотах крыши (§5.В).
- **Город = хаб (купол).** Стационарный. Казна, склад, переработка, найм, **все улучшения** — как у DK между волнами. Его осаждают ночью → якорь обороны по построению. Диегетика ✅: гномье поселение + Хартия («под крыло»).
- **Гномы = малыши-рабсила, расходник.** Добывают/пылесосят шматки, воюют в данжах, крутят переработку в хабе. Тратятся (солдат = потраченный гном ✅), гибнут, восполняются наймом (кап 7 ✅).
- **Рука = манипулятор игрока, НЕ сущность.** БЬЁТ жилы (стоячие пук-пук / ходячие — шлёп на ходу; сама НЕ таскает — носят гномы), кастует/дэш-формы ✅, командует артелью ✅, решает пазлы ✅, ставит лучника ✅.

**Тугость (ядро):** днём — жадный выезд (как далеко от хаба, как полон трюм, сколько гномов рискнуть); ночью — оборона хаба тем, что успел навесить и расставить.

## 3. Луп

### 3.1 Микро (секунды) — рука жонглирует
Мир: пук-пук жилу / ткнуть артель / поставить карту. Бой: каст / дэш-форма / передвинуть лучника / заткнуть прорыв. Пока рука в жиле — она не обороняет; DK-вычитание на внимании.

### 3.2 Мезо (день/ночь) — качели «выезд ↔ осада»
- **День (выезд):** башня выезжает от хаба к жилам → рука крошит → гномы пылесосят в трюм → трюм полон/солнце садится → домой → выгрузка в склад хаба → переработка → **улучшения и расстановка обороны** (вся setup-логистика — ДНЁМ). Бродячие банды = лёгкое давление в поле (✅ WaveDirector, день).
- **Ночь (осада хаба):** скелеты штурмуют ХАБ (✅ мелей у стен + рэйндж-залпы группами, дуга-тревога на банду). Башня обязана быть дома — обороняет тем, что навесила и расставила. **Рука ночью только воюет** (касты/дэш-формы); никакого микро-менеджмента — всё расставлено днём. Затяжная осада = экономическое кровотечение — уточнение причинности (в коде тревога V: смены ОСТАЮТСЯ в зданиях, прячутся только свободные ✅): переработка встаёт, когда игрок САМ снял гномов на оборону или осада ломает здания — кровотечение это выбор/урон, не автоматика. 🎯
- DK-уроки для волны (🎯, см. разбор §3): **weight-бюджет** вместо N врагов; сложность = f(прогресс, богатство игрока); `minRunWeight`-гейт вводит новые типы скелетов плавно; **rubber-band** по урону хабу; **анти-стойл эскалация** против кайта; два флага фазы (`wavepresent`/`wavebattle`); DiveWarner-телеграф по вектору атаки.

### 3.3 Макро (забег) — от хаба к хабу до Врат
Анфилада комнат (✅ rooms-геймлуп, туториал Room1–5, §5.22/5.26). Радиус выезда от хаба = ось жадности дня (дальше = жирнее жилы/данжи, дольше дорога домой). Выработал регион → **переход к следующему поселению-хабу = «спуск на слой глубже»** 🎯; по пути артефакты (Рецепт→кафедра огня, Аккумулятор→молния+шквал, Клад→+300🥉 — сюжет §5.28 ✅ утверждён). Дверные пазлы гейтят (✅ диод/провод/рычаг/мост-доска). Кульминация: **плата Вратам 900🥉 будит меха** (соло-apex, строго 1) + осаду со всех сторон (§5.28).

### 3.4 Мета (между забегами) ✅
Свитки → спеллы навсегда; чертежи → карты в колоду насовсем (`user://tower_profile.cfg`); кафедры = бафы пока стоят. = DK `unlockedElements`; кандидат на добор — их `gadgetToKeep` (перенос одного предмета в след. забег).

## 4. Экономика — труба капов (🎯 главный недострой)

**Проблема:** сейчас база = кран (авто-капание) → рисковать нечем. DK-урок: экономика = **пропускная труба**, каждый апгрейд расширяет одно горлышко.

**Типы жил:**
- **Стоячие** ✅ v1 2026-07-21 (`vein_boulder.gd`, `VeinBoulder` в level_rooms ×3 у хаба) — рука пук-пукает на месте (любой урон копится, каждые `damage_per_chunk` выбивает шматок-[ResourceOrb]) → шматки лежат → пока собирает МАГНИТ БАШНИ (орб сам летит в трюм, когда башня рядом и есть место; артель-пылесос = стадия 2). Визуал = общий язык залежей (`VeinVisuals`, тот же что маркеры грида). Исчерпана → рассыпается shatter'ом.
- **Ходячие** (часть жил ИЗНАЧАЛЬНО живые, бродят по полю) — рука должна **шлёпнуть на ходу**, чтобы расколоть → высыпается ресурс (лежит) → гномы собирают. Не пришлёпнул вовремя — **жила ушла, ресурс потерян** (чистая ставка на реакцию; просто уходит, без погони/пасфайндинга). Обычно жирнее стоячих (риск-ревард).

**Разделение труда — ПО СОСТОЯНИЮ цели, без хардкод-фильтра:** движется → **бьёт РУКА**; лежит → **несёт АРТЕЛЬ**. Рука вообще не таскает (отменяет прежнее «рука несёт один шматок»), гномы не гоняются за движущимся. Даёт руке непрерывную скилл-работу в добыче («не крякнул и стой») и DK-вычитание «пока бью жилы — не обороняю».
Переиспользование: ходячая жила = walk-AI скелета ✅; смак = импакт-слой §5.21 ✅; высыпавшийся ресурс + сбор гномами = ✅.

**Целевой поток:** жила (стоячая/ходячая) → **(1) рука бьёт** (пук-пук / шлёп по ходячей, быстро — нарочно обгоняет трубу) → лежачий ресурс → **(2) артель пылесосит** в трюм башни — кап: `свободные гномы × несёт за ходку × скорость` → **(3) трюм башни** = carry-кап кипера (ось прокачки в верфи) → **(4) выгрузка в склад хаба** (кап хранения ✅) → **(5) переработка в хабе** гномами в монеты/ману (гном-в-здании ✅ модель есть).

- **Рассинхронить капы** (крош > сбор > переработка): перекос = жонглирование; ровная труба = скучный кран.
- **Сырьё на земле тлеет** (таймер на дроп) = «не руби больше, чем унесёшь».
- Трюк шреддера DK: больше завал у станции → быстрее переработка (`coolDown = 1/level`).
- Тензия сбора: выгнал артель в поле = не на станциях, не на стене, уязвимы.
- **Валюта ✅:** единая монета, одометр 1🥇=10🥈=10🥉, цены одним номиналом. PAGE/FOOD ⛔ мертвы. Сырьё — вход переработки, НЕ «еда гномам».
- **Сейчас в коде** (замещается): горшки → монеты (✅ `pot.gd`), брёвна → казна +4🥉 (✅ `soldier_gnome.gd`).
- Прототипировать **стадию 2 первой** (пропускная артели) — она даёт весь фил.

## 5. Стройка и улучшения — ПЕРЕСБОРКА 2026-07-21 (🎯 новый канон)

### 5.А Ядра — ТРИ здания (замок ⛔ вырезан, якорь города = Верфь)
- **Верфь (2×2)** — ЦЕНТР и якорь грида, дом башни, ночная осада идёт на неё. Парковка чинит ✅. ВСЕ улучшения башни здесь: кап HP, броня, трюм, **+слот крыши** (главный дефицит). Найм артели тоже тут (кров при тревоге). «Нутро башни по срезам» = визуализация этих улучшений в окне срезов ✅ — НЕ вторая стройка.
- **Машинный цех (Г, 3 кл.)** — экс-институт: куёт БОЕВЫЕ МАШИНЫ по чертежам, продаёт узлы их веток, льёт энергию (экс-мана, механика та же ✅).
- **Плавильня (Г, 3 кл.)** — приёмник трубы (стадия 4, §4): гном плавит привезённое башней сырьё в монеты. Шахта-кран ⛔.

### 5.Б Ветки ядер = ПРИСТРОЙКИ-полимино 🎯
Узел ветки — не строчка меню, а фигура: купил кликом по ядру → втиснул в его зону-соседство (реюз сапорт-плейсмента ✅). Принадлежит ядру и дереву (зависимости, растущая цена, repeatable = ещё одна такая же). Видов мало, визуальный язык родителя (печь у плавильни, эллинг у верфи, котёл у цеха). Жирные пристройки берут гнома-смену ✅ (тугость артели). Ломаются осадой → узел гаснет до отстройки. Данные веток — decl-каталог (F5/DK upgrades.yaml).

### 5.В Машины на крыше 🎯 («магия = машины гномов»)
- Куются в цеху, **рука ставит в слот крыши** (глагол доказан: гарпун ✅, лучник ✅). Слотов мало: старт 1 → кап ~3 (верфь). «Полные трусы заклинаний» невозможны структурно.

### 6.1 Импакт — фичи улучшения (разбор HighFleet, 2026-07-21)
Что УЖЕ зрело (не трогать, не «изобретать заново»): hitstop per-enemy (`hit_stop.gd`, только боссы), tilt/squash (`hit_tilt.gd`), hit-flash (`hit_flash.gd`), отдача башни на выстрел (§5.21.3), trauma-тряска с зум-инвариансом (`camera_rig.gd`), shatter (`shatter_effect.gd`), knockback с resistance (`knockback_state.gd`), burn (`burn_patch.gd`). Урок HighFleet: выстрел = импульс в физ-тело → ломает КОНКРЕТНЫЙ кусок → рвёт связь → поджигает соседей → трясёт экран отдельным тюнимым слоем. Отсюда фичи (статус: ✅ есть / 🔌 есть, не подключено / 🎯 новое):

- **F1 — Направленный shatter по вектору удара 🔌→частично ✅ (2026-07-21).** Механизм ЕСТЬ (`ShatterEffect.spawn burst_dir/power`). СЛЭМ теперь передаёт `hit_dir` в `try_damage` → скелетий death-shatter и жила (веер осколков-шматков конусом ОТ руки + направленное рассыпание) направленные. Осталось: прокинуть hit_dir из fireball/firestorm/дэша в их try_damage-вызовы.
- **F2 — Единая шкала «impact_force» 🎯 (частично ✅).** Сейчас разрозненно: shake ∝ типу снаряда (фаербол 0.2/шквал 0.18/супер), knockback ∝ resistance, а recoil-kick ФИКС (0.22м). Свести в одну силу удара, от которой едут recoil + shake + `power` осколков. HighFleet: всё от массы стрелка/цели.
- **F3 — Локальный урон + отстрел частей для КРУПНЫХ 🎯.** Скелетам не надо (collider=damageable, одиночные). Мех/гигант/**башня** — модульные: под-части с HP, отстрел (ствол/нога/навес) → часть отваливается через F1. HighFleet: дерево Body+Joint. Для башни ложится на навороты верфи (каждый навес = повреждаемая часть). Крупная работа — заложить архитектурно, делать после трубы.
- **F4 — Огонь ползёт + детонация 🎯 (база ✅ `burn_patch`).** HighFleet `m_burn_hp` = порог + распространение. Шаг 1: горящий враг поджигает вплотную стоящего соседа (кормит AOE-фантазию шквала/фаербола). Шаг 2: «взрывоопасные» цели (бочка/погреб/носитель) детонируют цепью. Скоуп держать узким.
- **F5 — Импакт-дайлы → в ДАННЫЕ, не константы 🎯.** Твой принцип «твикать в редакторе» × DK data-driven × HighFleet `values.txt`. Собрать `MAX_TILT_DEG`/`_RECOIL_KICK`/shake-пресеты/shatter-импульсы в один `ImpactProfile.tres` — крутить фидбек без кода.
- **F6 — Ударная волна-кольцо на тяжёлых событиях ✅→расширить.** Шейдер есть (`slam_distortion.gdshader` — ripple+хроматика, уже на слэме руки). Триггерить на редких пиках: супер, топот меха, смерть башни/гиганта. HighFleet: Shockwave + Heat-haze как пост-слой.

Приоритет: **F1 (частично ✅ 2026-07-21: слэм передаёт hit_dir; осталось из спеллов/дэша) → F5 (выносит тюнинг в данные) → F2 → F4 шаг1 → F6 → F3 (крупное; с возвратом срез-стройки §5 модульная башня = та же архитектура — проектировать вместе).** F1/F2/F5/F6 почти целиком переиспользование; новьё по сути F3 и F4.

### 6.2 Пещера: отряд гномов = дрифт-машинка (✅ песочница 2026-07-23)

**Зачем.** Фан «игры за гномов» не в RTS-микро (солдат-дубль игрока на поле = балласт): гном ценен тем, чего нет у руки и башни — мал (проёмы/данжи, башня геометрически не пролезает), автономен, смертен-как-валюта. Данж-эпизод = смена роли: автошутер отрядом с управлением мышью. Анти-VS: у героя VS одно решение (позиция) — у нас поверх рулёжки решения «стоять/ехать», «дрифт/грип», «газ/тормоз».

**Модель «тачка» (финал итераций одного дня, все ручки — exports на корне сцены):**
- **ЛКМ (держать) = газ и руль.** Отряд течёт за курсором-поводком (кламп 7м → 10м с разгоном — запас на замах). Отпустил — ЮЗ по инерции (докатываются, стреляя), ёж собирается на месте остановки. Нажал снова — юз мгновенно перебивается: ЛКМ = педаль.
- **Разгон** = только скорость (3.5с непрерывного бега → +70%), сцепление НЕ трогает. Базовый руль слегка плывёт (grip 5.5) — контроль добирается тормозом.
- **Дрифт = СОБЫТИЕ, не состояние-фейд:** на скорости резкий увод точки (угол ≥55°) → весь отряд СИНХРОННО в скольжение («одна машинка»): сцепление разом падает (5.5→2.6), отдача пылью, полный наклон, линия горит; дуга довернулась ≤18° → синхронный выход. Гистерезис держит дугу.
- **Карв с трением:** в скольжении направление поворачивается, величина сходится к желаемой (scrub 1.6) — момент живёт (орбиты вокруг целей БЕЗ доп. кнопок), разлёта нет. **Сцепка строя** (pull, кап скорости обязателен) — куча дрифтит пачкой, не напарывается на ловушки.
- **ПКМ = тормоз/ручник:** ход и разгон стекают вместе, руление живёт (ручник-поворот = вход в дрифт с тормозом), осаживание корпусом, пыль.
- **Бой из движения:** стоя — кольцо-«ёж» на 360° (конусы наружу); на бегу — огонь только вперёд по курсу; захват цели: зацепил конусом → держит на 360° в attack_range (орбита-полив). Выстрел не стопит и не сбивает разгон.
- **Читаемость:** наклон корпуса = скольжение/разгон, пыль = «визг шин» (только дрифт/тормоз), линия поводка наливается цветом, хвост кометы на бегу ↔ ёж на стопе (сглаженный разворот).

**Песочница:** `scenes/dungeon_sandbox.tscn` + `scripts/dungeon_sandbox.gd` — ВНЕ геймлупа rooms; юниты штатные (ArcherSoldier+Squad+Skeleton, новых не заведено; аддитивные exports SoldierGnome с дефолтом 0 = основная игра не тронута). Баланс песочницы: урон лучника 8–12 (~3 выстрела на скелета), волны 4+2N. Грабли тест-сцен: LOD юнитов выключать до add_child (FAR = полёт сквозь стены), комнату держать вдали от мирового нуля (пустой навмеш снапит в (0,0,0)).

**⛔ Отвергнуто в итерациях (не возвращать):** плавный grip-фейд по разгону («дрифт подкрадывается»), верёвочный свинг с доворотом в касательную («противоестественно»), карв без трения (разлёт), резкий стоп на отпускании (сменён юзом с приходом «тачки»), ПКМ-пин (код спит).

**🎯 Очередь слоёв:** груз-разоружение (несущий лут не стреляет — жадность ослабляет залп), сбитые-vs-смерть (v1 = простая смерть; «сбитые с подбором» — если потери злят), формации по обводке фигур (язык дэш-форм; «ёж/хвост» уже эмерджентная база), интеграция в данжи rooms.
