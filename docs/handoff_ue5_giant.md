# ТЗ: Скелет-Гигант (босс-дуэлянт) — разработка на UE5

**Статус:** техническое задание для разработчика. Референс-реализация — Godot-прототип
(`scripts/skeleton_giant.gd`, `scenes/skeleton_giant.tscn`), поведение выверено плейтестами
2026-07-05..07. Все числа в этом документе — актуальные значения прототипа; это стартовые
значения тюнинга, НЕ константы (см. §12 «Ручки»).

---

## 1. Роль и фантазия боя

Гигант — **первый босс** и ось акта Room4: победа над ним открывает социальную цепочку
(Хартия → найм артели → путь дальше). Формула боя — **матадор-дуэль**:

> Гигант заряжает рывок-таран → игрок уворачивается → промахнувшийся гигант
> **оглушается** (стоит столбом, светится синим) → окно безопасного наказания →
> ровно **3 супер-рывка игрока = смерть**.

Ключевые принципы:
- **Каждая атака телеграфируется.** Никаких мгновенных ударов, в том числе вплотную.
- **Гигант — таран для всех**: его заряд давит и собственную мелочь на пути.
- **Бой сам себя кормит**: призывы на порогах HP — это и угроза, и мана-топливо игроку
  (труп скелета = орб = мана на супер-рывок).
- **Смерть — кульминация**: взрыв, испепеляющий нежить вокруг, слоумо-бит, салют орбов.

## 2. Базовые параметры

| Параметр | Значение | Примечание |
|---|---|---|
| HP | **750** | ровно 3 супер-рывка игрока (250). **БЕЗ случайного разброса** — обещание «3 удара» держится точно |
| Урон заряда по башне | 84 | ±20% допустимо |
| Скорость ходьбы | 3.2 | ±15% допустимо |
| Радиус зрения | 30 | |
| Дистанция начала заряда | 11 | attack_range |
| Замах перед зарядом | **0.7с — всегда** | ⚠ и вплотную тоже! Сокращение замаха на ближней дистанции (стандартное для рядовых) гиганту **запрещено** — это был источник «спама атак» |
| Кулдаун между зарядами | 2.4с | ±15% допустимо |
| Сопротивление отбросу | ×0.2 | танк: внешние толчки гасятся в 5 раз |
| Капсула | r 0.5, h 4.4 | заметно крупнее рядового скелета |

Пер-спавн вариация (если реализуете): замах/скорость/кулдаун — да; **HP — никогда**.

## 3. Стейт-машина

```
WANDER ──(аггро: башня вошла в комнату)──► ROAR (подача, один раз)
                                              │
              ┌───────────────────────────────▼──────────────┐
              │                    APPROACH                   │◄────────────┐
              │  идёт к башне; прерывается DODGE на снаряды   │             │
              └──────────────┬────────────────────────────────┘             │
                             │ дистанция ≤ 11                               │
                       WINDUP (0.7с телеграф)                               │
                             │                                              │
                       CHARGE (рывок 30 м/с, 0.45с, ≤13.5 м)                │
                       по пути: TRAMPLE своих (§6.2)                        │
                   ┌─────────┴─────────┐                                    │
             контакт с башней       промах                                  │
                   │                   │                                    │
             HIT: урон 84        SHOCKWAVE (§6.3)                           │
             SHOCKWAVE (§6.3)          │                                    │
                   │              STUNNED (1.5с, §7)                        │
             COOLDOWN (2.4с)           │                                    │
                   └───────────────────┴────────────────────────────────────┘
HP ⅔ / ⅓ (в любом состоянии): SUMMON + PHASE-UP (§9)
HP = 0: DEATH → взрыв (§10)
```

UE5-рекомендация: **StateTree** (или BT с центральным Blackboard). Заряд — через
**Root Motion / Motion Warping** к зафиксированной в момент старта точке (направление
на башню фиксируется на старте заряда и НЕ корректируется в полёте — иначе уворот
невозможен, а он ядро дуэли).

---

## 4. Аггро и room-gate

- Гигант **привязан к комнате**: прямоугольник `room_center` / `room_size` (XZ).
  Цель = башня **только когда она внутри прямоугольника** (по bounds, НЕ по дистанции —
  иначе аггрится «сквозь стену» на башню в соседней комнате).
- Гистерезис де-аггра: башня вышла за комнату + буфер 4 м → гигант бросает цель,
  возвращается в wander по комнате.
- Без заданного room-gate (спавн волной/читом): будится в радиусе 60 м.

---

## 5. Рёв-подача (первый аггро, один раз за жизнь)

Как только башня впервые стала целью:
1. **Баннер** сверху экрана: «ВНИМАНИЕ! ГИГАНТ» — пульсирует ~3с.
2. **Сильный камера-шейк** (0.7 в наших единицах — «самый тяжёлый вне смертей»).
3. Пыль + электрический разряд у ног гиганта (рёв-поза в анимации приветствуется).
4. **Аггро всей комнаты**: вся рядовая нежить в границах комнаты получает цель = башня.
   Бой плотный с первой секунды; мелочь = мана-фарм прямо в бою.

---

## 6. Атака: заряд-таран

### 6.1 Основной цикл
- Телеграф-замах 0.7с (анимация «присел-завёлся», читается издалека).
- Рывок 30 м/с × 0.45с (макс ~13.5 м) по зафиксированному направлению.
- Контакт с башней (дистанция ≤ 2.8): **один** удар 84, рывок гасится.

### 6.2 Trample — заряд давит своих
Во время заряда рядовые скелеты в радиусе **1.8 м** от гиганта:
- получают **60 урона** (рядовым — смерть) + отброс по направлению заряда (+вверх);
- каждый скелет — **один раз за заряд** (сет задетых чистится на старте заряда);
- других гигантов не задевает.

### 6.3 Shockwave — финиш заряда
Любое окончание заряда (попал ИЛИ промазал):
- расширяющееся кольцо на земле (оранжевое) + пыль + короткий шейк;
- рядовая нежить в радиусе **4.5 м** расшвыривается физическим отбросом (импульс 9 + вверх);
- **урона кольцо не наносит**, башню не толкает (башня наказана самим контактом заряда).

---

## 7. Стан-матадор (окно наказания)

**Условие:** заряд закончился БЕЗ контакта с башней (игрок увернулся).

- Длительность **1.5с** (впритык на супер-рывок игрока: hold 0.3 + прицел в слоумо —
  окно ощущается наградой за уворот, не бесплатной паузой).
- Гигант **стоит столбом**: AI/движение/уворот полностью заморожены.
- **Визуал — ключевой сигнал**: тело светится **синим** (у нас — подмена материала с
  синей эмиссией; в UE — материал/оверлей + нотификация в UI по вкусу). Игрок должен
  считывать «ОКНО ОТКРЫТО, БЕЙ» боковым зрением.
- В стане гигант не уворачивается → дальнобойные снаряды игрока влетают гарантированно.
- Выход из стана: материал возвращается, FSM → APPROACH.

---

## 8. Уворот от снарядов (dodge-duelist)

Вне заряда и стана гигант **уклоняется** от снарядов игрока:
- детект: снаряд игрока в радиусе 8 м;
- реакция: короткий рывок вбок (16 м/с × 0.18с ≈ 2.9 м), направление — перпендикуляр
  от снаряда, с поправкой «не уклоняться ПОД башню»;
- кулдаун уворота 0.7с.

Баланс-смысл: одиночный точечный снаряд (наша «Искра», impact-радиус 1.5) на бегу
**мажет**; AoE-заклинания радиусом перекрывают уворот — это их контрплей.

---

## 9. Правила урона и фазы

### 9.1 Урон по гиганту
- **Обычный таран башни гиганта НЕ берёт** (семантика «только супер-рывок», как у
  красной двери) — реализовано группой/тегом `super_dash_only`, не хардкодом по классу.
- Супер-рывок игрока: 250 урона → **ровно 3 попадания = смерть**.
- Заклинания/AoE бьют в полную силу. Множитель `active_damage_factor` (сейчас **1.0** —
  уязвим всегда) оставить ручкой: <1.0 включает «броню активного» (полный урон только
  в стане). Плейтест выбрал 1.0.
- Хитстоп на попадании по гиганту: усиленный (наш SUPER 0.16с с рефрактором 0.35с —
  серия попаданий не держит его в стан-локе).

### 9.2 Фазы (пороги ⅔ и ⅓ HP, срабатывают на пересечении вниз)
На КАЖДОМ пороге, одновременно:
1. **Призыв**: 6 рядовых скелетов кольцом r=4 вокруг гиганта, с рёвом/разрядом/шейком.
   Урон призванных занижен (у нас 3 — «раздражают, не убивают»): они топливо для маны
   игрока не меньше, чем угроза. **На смертельном ударе призыва НЕТ.**
2. **Эскалация**: скорость ходьбы ×1.3, кулдаун заряда ×0.85 (компаундится:
   2.4 → 2.04 → 1.73с). Финальная треть боя — самая горячая.

---

## 10. Смерть

1. Стандартный death-FX скелета (шаттер-осколки, гигантский масштаб).
2. **Взрыв**: радиус 6, **120 урона ТОЛЬКО по нежити** (итерация по группе скелетов);
   башня, гномы, постройки — не задеты. Отброс выживших (импульс 10 + вверх).
   Смысловая сцена: финальный удар испепеляет призванную мелочь вокруг, орбы с неё
   салютом летят в башню.
3. Кульминация тайм-фила (у нас глобальная политика на смерть «тяжёлых»):
   слоумо-бит ×0.3 на 0.25с + шейк + тёплая вспышка экрана.
4. Квест-хук: смерть гиганта = «доброе дело» (флаг для диалога гномов) — сигнал
   наружу, боссу об этом знать не нужно.

---

## 11. Импакт супер-тарана игрока (требование к связке башня↔гигант)

Момент попадания супер-рывка башни по гиганту — **кульминация**, пакет обязателен:
- слоумо-бит ×0.25 на 0.18с (реального времени);
- тяжёлый камера-шейк (0.8) + **синеватая вспышка экрана** (цвет супер-рывка);
- в точке контакта: сноп искр + пыль + расширяющееся синее кольцо;
- **отброс-впечатывание** гиганта: сильный импульс по направлению рывка — его
  танковый резист (×0.2) превращает удар в короткий читаемый слайд (~0.5 м);
- по рядовой мелочи тот же супер-рывок бьёт БЕЗ бита (иначе каша слоумо на толпе).

## 13. Чек-лист приёмки (плейтест-критерии)

- [ ] Вход в комнату → баннер + шейк + вся мелочь комнаты бежит к башне.
- [ ] Заряд ВСЕГДА телеграфируется 0.7с, в том числе в упор. Спама атак вплотную нет.
- [ ] Направление заряда фиксируется на старте — уворот шагом вбок стабильно работает.
- [ ] Промах заряда → гигант 1.5с стоит и светится синим; в это окно успевается супер-рывок.
- [ ] В стане дальнобойные снаряды игрока не уворачиваются, влетают все.
- [ ] Заряд убивает рядовых скелетов на своём пути; на финише мелочь расшвыривает кольцом.
- [ ] Ровно 3 супер-рывка убивают гиганта (без разброса HP).
- [ ] Обычный (не супер) таран башни урона гиганту не наносит.
- [ ] На ⅔ и ⅓ HP: пачка из 6 скелетов кольцом + гигант заметно ускоряется.
- [ ] На смертельном ударе пачка НЕ спавнится.
- [ ] Смерть: взрыв убивает рядовую нежить в радиусе, башня и гномы целы; слоумо-бит + шейк.
- [ ] Попадание супер-рывка по гиганту — слоумо-бит, вспышка, кольцо, гигант проезжает от удара.
- [ ] Гигант не преследует башню за пределами своей комнаты (+4 м буфер).
