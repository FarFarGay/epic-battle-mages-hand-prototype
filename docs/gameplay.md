# EBM — Геймплейный документ

**Статус:** живой gameplay-документ (player-facing дополнение к техническому `SPEC.md`).
**Движок/сцена:** Godot 4.6.2, main = `scenes/level_rooms.tscn` (комнатный данж-геймлуп).
**Обновлён:** 2026-07-06.

> Документ описывает геймплей с точки зрения **игрока и дизайна** (что делаешь, зачем, как ощущается, какие числа крутить и где). Реализация и low-level — в `SPEC.md`. Числа сверены с кодом; в таблицах указан файл-источник для тюнинга.

---

## 1. Цель и фантазия

Сага о метровых гномах-пограничниках, которые выживают в суровом мире средневековых войн людей и орд классической нежити, управляя семиметровой гранитной боевой башней на исполинских человеческих ногах, которая служит им мобильным аванпостом и главным оружием в тактических битвах.


---

## 2. Управление

| Ввод | Действие | Action (input map) |
|---|---|---|
| **W/A/S/D** | Движение башни | `move_forward/back/left/right` |
| **Мышь** | Позиция руки (курсор в мире) | — |
| **ЛКМ** | Рука: схватить/бросить предмет; вблизи домика — заговорить | `hand_grab` |
| **ПКМ** | Рука: каст активного заклинания (Искра) | `hand_action` |
| **Space (тап)** | Рывок башни | `dash` |
| **Space (удержать)** | Прицел супер-рывка (слоумо) → **ПКМ** = выполнить | `dash` + `hand_action` |
| **Q** | Парирование | `parry` |
| **1–5** | Выбор заклинания: трей нумерует по РАЗБЛОКИРОВАННЫМ (только Искра → она = `1`) | `equip_*` |
| **J** | Журнал | `ui_journal` |


**Два инструмента — две роли:** башня = грубая сила и перемещение (WASD, рывки, таран); рука = точность (магия по ПКМ, захват предметов, пазлы, диалоги). Камера изометрическая, следует за башней.

## 3. Башня — параметры

| Параметр | Значение | Источник |
|---|---|---|
| Скорость движения | 8 | `tower.gd` `move_speed` |
| Макс. мана | 100 | `max_mana` |
| Старт маны | **0** | `mana` |
| Реген маны | 0.8 / сек | `mana_regen_rate` |
| Отталкивание врагов (таран) | ×1.5 скор., 0.2с | `enemy_push_*` |

Башня физически отталкивает скелетов корпусом, давит горшки, ломает красную дверь рывком. При гибели — взрыв (радиус 7, урон 220) как защитный «последний аргумент».

### 3.1. Обычный рывок (Space, тап)
Короткий бросок вперёд. Урон по задетым врагам.

| Параметр | Значение |
|---|---|
| Скорость / длительность | 22 / 0.24с |
| Кулдаун | 0.8с |
| Урон | 100 |

### 3.2. Супер-рывок (удержать Space → ПКМ)
Удерживаешь Space ≥ 0.3с → **время замедляется** (slowmo ×0.15), у руки появляется **синий прицел-маркер**; **ПКМ** запускает мощный рывок к маркеру. Дальше, сильнее, стоит **50 маны**. Главный инструмент пролома: **разносит красную дверь на физические куски**, легко добивает гиганта.

| Параметр | Значение | Источник |
|---|---|---|
| Порог удержания | 0.3с | `tower.gd` `super_dash_hold_threshold` |
| Слоумо | ×0.15 | `super_dash_time_scale` |
| Скорость / длительность | 34 / 0.34с | `super_dash_speed/duration` |
| Урон | 250 | `super_dash_damage` |
| Стоимость | 50 маны | `super_dash_mana_cost` |
| Радиус маркера | 1.4 | `super_dash_marker_radius` |

> Во время прицела супер-рывка ПКМ делает **только** коммит рывка (Искрой не стреляет) — намеренно, чтобы не было случайного каста.

### 3.3. Парирование (Q)
Кулдаун 1.5с, радиус 6, урон по скелетам 100 (`parry_*`).

## 4. Мана и заклинания

**Мана** — топливо магии и супер-рывка. Стартуешь с **0**, медленно регенерится (0.8/с), но основной приток — **сбор XP-орбов со скелетов**: башня вакуумит орб при сближении, +5 маны за орб (1 «манатанк» = 100 маны = ~20 скелетов).

### 4.1. Искра (единственное доступное заклинание)
Быстрый дешёвый снаряд по ПКМ. Помимо урона — **активирует «искровые» цели** (`spark_target`): синие диоды-пазлы и горшки.

| Параметр (ур.1) | Значение | Источник |
|---|---|---|
| Урон | 35 | `spell_system.gd` `spark` |
| Кулдаун | 0.15с |  |
| Мана за каст | 3 |  |
| Радиус импакта | 1.5 |  |

### 4.2. Прочие заклинания — заблокированы
В каталоге есть Огнешар (1), Огнебуря (2), Мины (3), Мороз (4), «Великий удар»/super (E), но все **закрыты** (`unlocked_by_default=false`) и убраны из трея. Сейчас у игрока только Искра — фокус на рывках и таране. (Параметры в `spell_system.gd` — на будущее.)

## 5. Враги

### 5.1. Скелет (рядовой)
Базовый melee-противник. В комнатах **без целей** (нет лагеря/гномов рядом) — просто **бродят** по навмешу (не лезут в стены — выбор точек проверяется на достижимость). Завидев башню/отряд — догоняют и бьют.

| Параметр | Значение | Источник |
|---|---|---|
| HP | ~30 | `enemy.gd` `hp` |
| Урон / дистанция | 8 / 1.5 | `attack_damage/range` |
| Замах | 0.4с | `attack_windup` |
| Радиус зрения | 12 | `skeleton.gd` `vision_radius` |
| Скорость блуждания | 1.2 | `wander_speed` |
| Дроп | 1 XP-орб (+5 маны) | — |

### 5.2. Скелет-Гигант (босс комнаты)
Первый настоящий босс-бой (Room4). Заезжаешь в комнату → **рёв-подача**: пульсирующий баннер, тряска камеры, и **вся мелочь комнаты бросается на башню** (давление с первой секунды; мелочь = мана-фарм прямо в бою). Правило боя: **гиганта ранит только супер-рывок** (обычный таран — нет; Искра бьёт, но он уворачивается от снарядов рывком вбок).

**Матадор-цикл:** гигант бьёт зарядом-разбегом (замах всегда полный 0.7с — вблизи не срезается); увернулся → гигант **промахивается и оглушается на 1.5с** (стоит столбом, светится синим) — окно безопасного наказания: супер-рывок в упор и/или Искра-спам (в стане не уворачивается). Заряд **давит его же мелочь по пути** (таран для всех), финиш заряда — **ударная волна** (мелочь разлетается). На ⅔ и ⅓ HP гигант **призывает пачку из 6 скелетов** и **звереет** — бой разгоняется к финалу. Ровно **3 супер-рывка = смерть**, и на смерти гигант **взрывается**: испепеляет нежить в радиусе 6 (башня и гномы не задеты), орбы с испепелённых — мана-салют победителю.

| Параметр | Значение | Источник |
|---|---|---|
| HP | 750 (= 3 супер-рывка) | `skeleton_giant.tscn` `hp` |
| Урон заряда | 84 | `attack_damage` |
| Скорость / кулдаун заряда | 3.2 / 2.4с (по фазам ×1.3 / ×0.85) | `move_speed`, `attack_cooldown` |
| Замах (всегда, и вплотную) | 0.7с | `attack_windup`, `attack_windup_point_blank` |
| Таран мелочи в заряде | 60 урона, r=1.8 | `charge_trample_*` |
| Взрыв на смерти | 120 урона нежити, r=6 | `death_explosion_*` |
| Стан промаха | 1.5с | `skeleton_giant.gd` `miss_stun_duration` |
| Призыв на порогах HP | 6 скелетов, кольцо r=4 | `summon_count` |
| Шоквейв заряда | r=4.5, мелочь разлетается | `charge_shockwave_*` |
| Room-gate | `room_center` / `room_size` | аггр только когда башня в комнате |

Убийство гиганта — **ключевое «доброе дело»** для гномов (см. §8–9) и ось акта Room4: победа → Хартия → торговля → отряд → гном-рычаг → дверь дальше.

## 6. Комнаты, двери, пазлы

Уровень — **анфилада комнат вдоль +Z**, разделённых дверями-преградами. Проходы широкие (≥3× ширины башни), кроме одного намеренно узкого. С 2026-07-05 анфилада работает как **обучающая арка «одна комната = один глагол»**: тело башни → рука-предмет → рука+цепь → бой/социум → командование.

| Комната | Урок | Содержимое | Выход / преграда |
|---|---|---|---|
| **Room1** (пещера, старт) | Тело башни | Старт в **тёмном тоннеле** (движение) → полость: 6 одиночек у устья (дэш) + ~72 скелета (мана с орбов) | **Красная дверь** — только **супер-рывок** (50 маны — надо нафармить) = «выход из пещеры» |
| **Room2** | Рука-предмет | **Пропасть поперёк** пути, мосток на ближнем берегу, 8 скелетов | Схвати мосток (ЛКМ) → положи поперёк → **металл-дверь**: Искрой по синему диоду |
| **Room3** | Рука + цепь | Цепь растянута по комнате с **разрывом**: гнездо в центре, диод-ретранслятор лежит в стороне, ~20 скелетов | Вставь ретранслятор в гнездо → Искра по синему → ток насквозь → **рычаг рукой** → металл-дверь. Пустое гнездо = «пшик» и цепь сбрасывается (бей снова — софтлока нет) |
| **Room4** | Первый бой + социум | Гномьи домики, **Скелет-Гигант** (босс, см. §5.2) | После победы: Искра по зап. диоду → ток → гном-рычаг в RoomWest ожил → **пошли артель** (узкий проём 1.6м — «башня не лезет, гном да») → металл-дверь |
| **RoomWest** | Кооп башня↔гном | малая комната 14×14, гном-рычаг за узким проёмом | — |
| **Room5** (финал) | Командование + заработок | **«Работа под угрозой»**: роща из 3 деревьев, готового моста НЕТ; рубка шумит → волны с флангов; 10 скелетов-арьергард за пропастью | Сданные брёвна = монеты (+4🥉/шт) → накопи 30🥉 → палитра «Стройка» → блюпринт **«Мостки»** (не на пропасти!) → готовую доску рука кладёт поперёк |

**Обучение — подсказки.** По уровню расставлены триггеры подсказок (плашка внизу экрана): зоны («WASD — движение», «ПРОБЕЛ — рывок»), события (первая мана, мана ≥ 50 → супер-рывок), шаги пазлов («цепь замкнута — бей Искрой и дёргай рычаг», «гигант пал — Искрой по диоду», «рычаг ожил — пошли артель»). Умные: «нужен мост» не показывается, если мост уже лежит.

**Двери:**
- **Красная** (`room_door.gd`) — мягкая преграда: только **супер-рывок** проламывает (физический шаттер + перепечка навмеша).
- **Металлические** (`metal_door.gd`) — рывок-непробиваемы, открываются **только пазлом** (уезжают вниз).

**Пазлы** — «магия как ключ»: Искра не просто оружие, а **инструмент активации** (искровые цели). Цепочка Room3 — трёхтактный пазл «поставь (рука) → запитай (Искра) → дёрни (рука)».

**Дизайн узкого проёма (1.6м):** разделяет «куда может башня» и «куда может только гном/рука» — задел под механики, где отряд/рука действуют там, где громоздкая башня не пройдёт.

### 6.1. Рабочие и стройка — направленная модель «ткни область → делай тут»

**7 гномов-рабочих живут в башне с самого старта** (докупка у домика доливает до капа 7). Рабочие **не воюют** (утилита). Главная команда — **«Идти сюда»** (area-клик ЛКМ): тыкаешь область на штуку, и они идут делать **ровно её**:

| Ткнул областью на… | Рабочие |
|---|---|
| **Дерево/ресурс** | рубят и складывают на **склад башни** (запас виден в HUD) |
| **Блюпринт стройки** | берут ресурс **со склада**, несут, кладут — строят |
| **Горшок** | разбивают (золото) |
| **Рычаг** | переключают |
| **Башню (повреждённую)** | чинят |
| **Пустое место** | просто идут туда |

**Мост через пропасть = ГИБРИД стройки и руки (Room5 «работа под угрозой», 2026-07-07):**
1. **Прикажи рубить** — карточка артели → «Идти сюда» → клик по дереву (башня должна стоять рядом: зона добычи 16 м). Рабочие рубят и носят брёвна к башне — **каждое сданное бревно продаётся казне (+4🥉)**: рубка = заработок, гном с бревном = живые деньги в пути.
2. **Шум будит скелетов** — первая волна на первый удар топора, дальше волна за каждое срубленное дерево (с флангов). Защищай свой доход; жарко — «В башню» прячет артель.
3. **Накопи 30🥉 → построй доску**: карточка артели → «🔨 Стройка» → «🌉 Мостки» (30🥉), блюпринт ставится где угодно, **кроме самой пропасти** (силуэт краснеет) — стройка не умеет «через дыру». Доска самостроится.
4. **Готовую доску кладёт РУКА**: схвати (ЛКМ), положи поперёк пропасти — защёлк, проход открыт. **Сама пропасть не исчезает** — проход только по мосту; доску можно забрать обратно. Гномы производят вещь — рука её применяет.

**Ремонт башни** — кнопка **«🔧 Ремонт башни»** или area-клик по башне: рабочие выходят, чинят ударами, отремонтировав — прячутся. **«В башню»** — спрятать артель ВНУТРЬ (неуязвимы) на опасный момент.

**Экономика стройки** — через склад: сперва насобирай дерева (#1), потом строй (#3, тратит со склада). Тыкнул блюпринт с пустым складом — рабочие ждут у башни, пока не появится дерево.

## 7. Экономика золота

### 7.1. Источник — горшки
Во всех комнатах расставлены **горшки**. Бьются от **действий башни** (в т.ч. просто проехать над горшком) и от **магии** (Искра). Из разбитого вылетают **монеты**, которые **магнитятся к башне** и капают в кошелёк.

| Параметр | Значение | Источник |
|---|---|---|
| Монет из горшка | 3 | `pot.gd` `coin_count` |
| Золота за монету | 10 | `gold_per_coin` |
| Горшков в сцене | ~10 | узел `Pots` |

**Смысл:** золото добывается «походя», разнося обстановку — постоянный мелкий приток без отдельного гринда, поощряет крушить всё подряд.

**Второй источник — работа артели (Room5+):** каждое сданное рабочим бревно продаётся казне (**+4🥉**, `soldier_gnome.gd` `WOOD_SALE_BRONZE`). Единая валюта: материалы существуют только в руках гнома, в казне всё — монеты.

### 7.2. Кошелёк
`GoldBank` — счётчик золота, независимый от старого лагеря. Текущее золото в HUD.

## 8. Награда за босса — Торговая Хартия

Убийство **Скелета-Гиганта** даёт не золото, а **возможность подписать Торговую Хартию** с поселением гномов.

- **Поселение** — гномьи домики в комнате (отдельная сущность). Один «говорящий» (рука подсвечивает, **ЛКМ** вблизи → диалог). Тон гнома — **Остин Пауэрс** («бэйби», «гру́вно», бархатный жилет).
- **Хартия** — документ с правилами честной торговли + **поле ввода имени игрока**. После подписи гномы **зовут тебя по имени** (`{name}` во всех репликах).
- **Гейт диалога:** до подписи — доступен путь к подписи, торговля скрыта; после — предложение подписать исчезает, появляется «Прикупить копейщиков».

**Смысл:** Хартия — «рукопожатие», ворота к торговле; подпись с именем — ритуал персонализации (гном из NPC становится «знакомым»).

## 9. Найм артели — один клик в диалоге

После подписи Хартии у Загадочного гнома появляется ветка **«Мне бы рабочие руки»** → пункт **«По рукам! Нанять артель — 5🥈»**. Клик — и всё: списалось 5🥈, **6 гномов идут в башню** (плашка подтверждает). Не хватает монет — плашка «артель стоит 5🥈 — разбей горшки»; артель уже полна — «в полном составе», деньги не тронуты.

| Параметр | Значение | Источник |
|---|---|---|
| Цена артели | 5🥈 (50 бронзы) | `gnome_house.gd` `ARTEL_PRICE_BRONZE` |
| Размер найма | +6 рабочих (кап 7) | `gnome_house.gd` `ARTEL_HIRE_COUNT` |

> История (2026-07-07): был диегетический **стол торга** с drag-and-drop монет и скидками за «добрые дела» — плейтест показал «не разберёшься, как покупать». Упрощено до одного пункта диалога с ценой в лейбле. Стол (`trade_ui.gd`) остался казармам города; «добрые дела» (`deeds_log.gd`) временно без потребителя — кандидат на скидки в городе.

## 10. Отряды

**Боевые отряды (копейщики/лучники)** в туториале недоступны — нанимаются в **казармах города** за монеты (гарнизон стен, призыв клавишей F за башню). Параметры/поведение: `SoldierSystem`, §SPEC 5.8.2.

### 10.1. Отряд рабочих — утилита, не бой

Главный отряд туториала — **бурые гномы-рабочие**: собирают ресурс, строят, чинят башню, бьют горшки/рычаги — всё **по направленному area-клику** (см. §6.1). **Не воюют** — символический урон только по дереву/стройке. **Старт — ОДИН рабочий в башне** (хватает на гном-рычаг Room4); после Хартии у гномов покупается **артель +6** (кап 7). Команда «В башню» прячет их внутрь (неуязвимы). Карточка рабочих: вкладки **«🔨 Стройка»** и **«🔧 Ремонт башни»**.

**Единая модель:** «Идти сюда» ЛКМ по цели → контекстное действие (дерево→собрать на склад, блюпринт→строить со склада, горшок→разбить, рычаг→переключить, башня→чинить). Никакой автономной беготни — делают ровно то, на что ткнул.

**Ремонт башни:** кнопка **«🔧 Ремонт башни»** или **area-клик по корпусу башни** — рабочие выходят, чинят замахом (рассинхрон), отремонтировав — прячутся. Башня — цель ремонта, лишь пока повреждена (`repair_per_hit=25` HP за удар). Копейщики тоже умеют прятаться: кнопка **«🏰 В башню»** (отдельно от «За башней»).

## 11. Полная петля

```
Пещера-туториал: движение → дэш → мана с орбов → СУПЕР-РЫВОК ломает красную дверь
        ↓
Комнаты-уроки руки: мосток через пропасть → ретранслятор чинит цепь → рычаг
        (попутно: горшки = ЗОЛОТО, скелеты = МАНА)
        ↓
Босс-комната: рёв → матадор-дуэль с Гигантом (уворот → стан → 3 супер-рывка)
        ↓
Гном-поселение: подписал Торговую Хартию (ввёл имя) → гномы знают тебя
        ↓
Диалог гнома: «Нанять артель — 5🥈» (один клик) → +6 рабочих, казна похудела
        ↓
Искра в диод → гном дёргает рычаг за узким проёмом → дверь в Room5
        ↓
«Работа под угрозой»: рубка рощи под волнами → брёвна продаются казне →
        30 бронзы → блюпринт «Мостки» → готовую доску рука кладёт поперёк → дальше
```

## 12. Открытые ветки и известные пробелы

- **⚠ Условие победы не подключено.** Формально есть `MatchGoal` (золото ≥ 1000 И «пройти врата»), но он читает экономику старого Camp → в room-режиме цель матча де-факто не финализирована. Альтернатива — нефть-замок (`oil_goal` → победа). Выбрать канон.
- **«Переплата золотом → рост репутации»** — слой к торговле; обсуждали, отложили.
- **Команды отряду** в room-режиме помимо эскорта (Hold/Defend) — каркас в `Squad` есть, UI-команд нет.
- **Полировка стола торга** — анимация падения монет, портрет гнома, дерево/пергамент.
- **Прочие добрые дела** как источники скидки (сейчас одно — убийство гиганта).
- **Прочие заклинания** — разблокировка/прогрессия (магазин Кафедры появляется со стройкой города).
- **Стройка** — мост вернулся в палитру как гибрид (2026-07-07): блюпринт «Мостки» за дерево производит ДОСКУ-предмет, финальная укладка — рукой. Остальная стройка = палитра полимино-города вокруг качалки-замка (SPEC §5.22.6/5.22.9, §5.26.6).
