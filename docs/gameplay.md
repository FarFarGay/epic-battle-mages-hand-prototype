# EBM — Геймплейный документ

**Статус:** живой gameplay-документ (player-facing дополнение к техническому `SPEC.md`).
**Движок/сцена:** Godot 4.6.2, main = `scenes/level_rooms.tscn` (комнатный данж-геймлуп).
**Обновлён:** 2026-06-18.

> Документ описывает геймплей с точки зрения **игрока и дизайна** (что делаешь, зачем, как ощущается, какие числа крутить и где). Реализация и low-level — в `SPEC.md`. Числа сверены с кодом; в таблицах указан файл-источник для тюнинга.

---

## 1. Цель и фантазия

Ты — **маг, ведущий живую боевую башню** сквозь подземелье гномьего города. Башня — твой аватар-герой: едет по комнатам, давит и расшвыривает врагов, копит ману со скелетов и золото с разбитой утвари. Параллельно у тебя есть **гигантская рука** — отдельный инструмент для тонкого взаимодействия (хватать предметы, дёргать рычаги, говорить с гномами, кастовать заклинания).

Прогрессия: пробиваешься из комнаты в комнату (двери-преграды и пазлы), валишь боссов-гигантов, налаживаешь торговлю с уцелевшим поселением гномов и **покупаешь отряд**, который едет с башней дальше. Фантазия — «маг + его шагающая крепость + наёмные гномы-копейщики против орд скелетов».

## 2. Управление

| Ввод | Действие | Action (input map) |
|---|---|---|
| **W/A/S/D** | Движение башни | `move_forward/back/left/right` |
| **Мышь** | Позиция руки (курсор в мире) | — |
| **ЛКМ** | Рука: схватить/бросить предмет; вблизи домика — заговорить | `hand_grab` |
| **ПКМ** | Рука: каст активного заклинания (Искра) | `hand_action` |
| **Space (тап)** | Рывок башни | `dash` |
| **Space (удержать)** | Прицел супер-рывка (слоумо) → **ПКМ** = выполнить | `dash` + `hand_action` |
| **Q** | Парирование | `parry` |
| **1–5** | Выбор заклинания (доступна только Искра = `5`) | `equip_*` |
| **E** | Спелл «Великий удар» — **заблокирован** (убран из трея) | `cast_super` |
| **J** | Журнал | `ui_journal` |

> R/F/C/V (`camp_toggle`/`caravan_halt_toggle`/`gnome_collect`/`gnome_alarm`) — легаси-контролы лагерного режима, в комнатном геймлупе не задействованы.

**Два инструмента — две роли:** башня = грубая сила и перемещение (WASD, рывки, таран); рука = точность (магия по ПКМ, захват предметов, пазлы, диалоги). Камера изометрическая, следует за башней.

## 3. Башня — параметры

| Параметр | Значение | Источник |
|---|---|---|
| Скорость движения | 8 | `tower.gd` `move_speed` |
| Макс. мана | 100 | `max_mana` |
| Старт маны | **0** | `mana` |
| Реген маны | 0.8 / сек | `mana_regen_rate` |
| Отталкивание врагов (таран) | ×1.5 скор., 0.2с | `enemy_push_*` |

Башня физически отталкивает скелетов корпусом, давит горшки, ломает красную дверь рывком. При гибели — взрыв (радиус 7, урон 220) как защитный «последний аргумент».

### 3.1. Обычный рывок (Space, тап)
Короткий бросок вперёд. Урон по задетым врагам.

| Параметр | Значение |
|---|---|
| Скорость / длительность | 22 / 0.24с |
| Кулдаун | 0.8с |
| Урон | 100 |

### 3.2. Супер-рывок (удержать Space → ПКМ)
Удерживаешь Space ≥ 0.3с → **время замедляется** (slowmo ×0.15), у руки появляется **синий прицел-маркер**; **ПКМ** запускает мощный рывок к маркеру. Дальше, сильнее, стоит **50 маны**. Главный инструмент пролома: **разносит красную дверь на физические куски**, легко добивает гиганта.

| Параметр | Значение | Источник |
|---|---|---|
| Порог удержания | 0.3с | `tower.gd` `super_dash_hold_threshold` |
| Слоумо | ×0.15 | `super_dash_time_scale` |
| Скорость / длительность | 34 / 0.34с | `super_dash_speed/duration` |
| Урон | 250 | `super_dash_damage` |
| Стоимость | 50 маны | `super_dash_mana_cost` |
| Радиус маркера | 2.2 | `super_dash_marker_radius` |

> Во время прицела супер-рывка ПКМ делает **только** коммит рывка (Искрой не стреляет) — намеренно, чтобы не было случайного каста.

### 3.3. Парирование (Q)
Кулдаун 1.5с, радиус 6, урон по скелетам 100 (`parry_*`).

## 4. Мана и заклинания

**Мана** — топливо магии и супер-рывка. Стартуешь с **0**, медленно регенерится (0.8/с), но основной приток — **сбор XP-орбов со скелетов**: башня вакуумит орб при сближении, +5 маны за орб (1 «манатанк» = 100 маны = ~20 скелетов).

### 4.1. Искра (единственное доступное заклинание)
Быстрый дешёвый снаряд по ПКМ. Помимо урона — **активирует «искровые» цели** (`spark_target`): синие диоды-пазлы и горшки.

| Параметр (ур.1) | Значение | Источник |
|---|---|---|
| Урон | 35 | `spell_system.gd` `spark` |
| Кулдаун | 0.15с |  |
| Мана за каст | 3 |  |
| Радиус импакта | 1.5 |  |

### 4.2. Прочие заклинания — заблокированы
В каталоге есть Огнешар (1), Огнебуря (2), Мины (3), Мороз (4), «Великий удар»/super (E), но все **закрыты** (`unlocked_by_default=false`) и убраны из трея. Сейчас у игрока только Искра — фокус на рывках и таране. (Параметры в `spell_system.gd` — на будущее.)

## 5. Враги

### 5.1. Скелет (рядовой)
Базовый melee-противник. В комнатах **без целей** (нет лагеря/гномов рядом) — просто **бродят** по навмешу (не лезут в стены — выбор точек проверяется на достижимость). Завидев башню/отряд — догоняют и бьют.

| Параметр | Значение | Источник |
|---|---|---|
| HP | ~30 | `enemy.gd` `hp` |
| Урон / дистанция | 8 / 1.5 | `attack_damage/range` |
| Замах | 0.4с | `attack_windup` |
| Радиус зрения | 12 | `skeleton.gd` `vision_radius` |
| Скорость блуждания | 1.2 | `wander_speed` |
| Дроп | 1 XP-орб (+5 маны) | — |

### 5.2. Скелет-Гигант (босс комнаты)
Толстый танк-дуэлянт. Идёт на башню, бьёт с разбега (charge) и **уклоняется** (dodge-duelist). **Аггрит только когда башня в его комнате** (room-gate: видит цель лишь внутри прямоугольника комнаты, не «сквозь стены» по дистанции) — заходишь к нему → начинается дуэль.

| Параметр | Значение | Источник |
|---|---|---|
| Урон удара | 84 | `skeleton_giant.tscn` `attack_damage` |
| Дистанция удара | 11 | `attack_range` |
| Замах | 0.7с | `attack_windup` |
| Скорость / зрение | 2.0 / 30 | `move_speed` / `vision_radius` |
| Charge (разбег) | 30 m/s | `skeleton_giant.gd` `charge_speed` |
| Room-gate | `room_center` / `room_size` |  |

Убийство гиганта — **ключевое «доброе дело»** для гномов (см. §8–9) и обычно главная боевая задача комнаты.

## 6. Комнаты, двери, пазлы

Уровень — **анфилада комнат вдоль +Z**, разделённых дверями-преградами. Проходы широкие (≥3× ширины башни), кроме одного намеренно узкого.

| Комната | Содержимое | Выход / преграда |
|---|---|---|
| **Room1** (старт) | Башня, ~80 скелетов (4 манатанка) | **Красная дверь** — ломается **супер-рывком** (разлетается на куски) |
| **Room2** | ~20 скелетов | **Металл-дверь** (рывок не берёт). Пазл: **Искрой по синему диоду** → дверь уезжает |
| **Room3** | пазл-цепь | Искра → **синий диод** → провод (бежит ток-бусина) → **красный диод** → **рычаг** (тянешь рукой) → металл-дверь |
| **Room4** | Гномьи домики (поселение), **Скелет-Гигант** (босс) | Металл-дверь; **узкий проём 1.6м** в зап. стену — «башня не лезет, гном да» → RoomWest |
| **RoomWest** | малая боковая комната 14×14 | — |

**Двери:**
- **Красная** (`room_door.gd`) — мягкая преграда: только **супер-рывок** проламывает (физический шаттер + перепечка навмеша).
- **Металлические** (`metal_door.gd`) — рывок-непробиваемы, открываются **только пазлом** (уезжают вниз).

**Пазлы** — «магия как ключ»: Искра не просто оружие, а **инструмент активации** (искровые цели). Цепочка в Room3 — мини-головоломка на последовательность (искра→ток→рычаг рукой).

**Дизайн узкого проёма (1.6м):** разделяет «куда может башня» и «куда может только гном/рука» — задел под механики, где отряд/рука действуют там, где громоздкая башня не пройдёт.

## 7. Экономика золота

### 7.1. Источник — горшки
Во всех комнатах расставлены **горшки**. Бьются от **действий башни** (в т.ч. просто проехать над горшком) и от **магии** (Искра). Из разбитого вылетают **монеты**, которые **магнитятся к башне** и капают в кошелёк.

| Параметр | Значение | Источник |
|---|---|---|
| Монет из горшка | 3 | `pot.gd` `coin_count` |
| Золота за монету | 10 | `gold_per_coin` |
| Горшков в сцене | ~10 | узел `Pots` |

**Смысл:** золото добывается «походя», разнося обстановку — постоянный мелкий приток без отдельного гринда, поощряет крушить всё подряд.

### 7.2. Кошелёк
`GoldBank` — счётчик золота, независимый от старого лагеря. Текущее золото в HUD.

## 8. Награда за босса — Торговая Хартия

Убийство **Скелета-Гиганта** даёт не золото, а **возможность подписать Торговую Хартию** с поселением гномов.

- **Поселение** — гномьи домики в комнате (отдельная сущность). Один «говорящий» (рука подсвечивает, **ЛКМ** вблизи → диалог). Тон гнома — **Остин Пауэрс** («бэйби», «гру́вно», бархатный жилет).
- **Хартия** — документ с правилами честной торговли + **поле ввода имени игрока**. После подписи гномы **зовут тебя по имени** (`{name}` во всех репликах).
- **Гейт диалога:** до подписи — доступен путь к подписи, торговля скрыта; после — предложение подписать исчезает, появляется «Прикупить копейщиков».

**Смысл:** Хартия — «рукопожатие», ворота к торговле; подпись с именем — ритуал персонализации (гном из NPC становится «знакомым»).

---

## 9. Торговля — стол с монетами

После подписи у гнома покупаешь отряд за **столом торга** — диегетический мини-процесс, а не сухой клик.

**На столе:** цена отряда; зелёная зона оплаты; кошелёк-стопка (drag-источник монет); добрые дела (реплики снизу = скидка); «**Купить за: N**».

**Как платишь:**
1. **Добрые дела** (напр. «Убил Гиганта») — клик включает **скидку** на цену; дело **тратится** при покупке.
2. **Монеты** — тащишь **drag-and-drop'ом** из кошелька в зону по одной, пока не наберёшь остаток (стол подсвечивается; клик по выложенной монете снимает её).
3. Набрал ≥ цены → **«Купить отряд»**.

| Параметр | Значение | Источник |
|---|---|---|
| Цена отряда | 200 | `trade_ui.gd` `base_price` |
| Скидка за «Убил Гиганта» | 150 | `deeds_log.gd` `GIANT_DEED_VALUE` |
| Номинал монеты | 25 | `trade_ui.gd` `COIN_VALUE` |
| Размер отряда | 3 | `trade_ui.gd` `squad_size` |

**Пример:** 200 → включил «Убил Гиганта» (−150) → платить **50** = 2 монеты.

**Смысл:** покупка тактильна (монетки руками); добрые дела создают мотив «делать хорошее гномам ради скидки» (с делом — 2 монеты, без — 8: видимый стимул).

> История: пробовали сложную модель (недоверие-стамина, две сообщающиеся шкалы золото↔отношения) — откатили к **простой скидке** (понятнее/быстрее).

## 10. Отряд — копейщики за башней

Покупка спавнит **отряд копейщиков** (`SoldierGnome`, `pikeman`) кольцом у башни; отряд **следует за башней** и врывается копьём в скелетов по пути.

- **Эскорт** — держат строй-кольцо вокруг башни, едут с ней.
- **Бой** — видят скелета в радиусе → charge-attack (разгон → замах → молниеносный выпад насквозь → откат).
- **Squad-абилка** — маркер заряда над отрядом; копится от убийств членами, по готовности ловит hand-slam (круговая push-волна).

| Параметр | Значение | Источник |
|---|---|---|
| Копейщиков | 3 | покупка (`squad_size`) |
| HP / урон / скорость | 30 / 22–32 / 2.2 | `SoldierSystem` `pikeman` |
| Радиус спавна у башни | 3.5 | `gnome_squad_spawner.gd` |

**Смысл:** отряд — растущая сила героя, **привязанная к башне, а не к статичной базе**; награда за весь цикл (босс→Хартия→торговля) материализуется в боевую поддержку, которая едет дальше.

## 11. Полная петля

```
Едешь по комнатам → крушишь горшки = ЗОЛОТО, бьёшь скелетов = МАНА,
                    магией/рычагами решаешь пазлы дверей
        ↓
Босс-комната: Скелет-Гигант → убил (супер-рывок / Искра / отряд)
        ↓
Гном-поселение: подписал Торговую Хартию (ввёл имя) → гномы знают тебя
        ↓
Стол торга: «Убил Гиганта» (скидка) + домонетил золотом → купил отряд
        ↓
Копейщики идут за башней, врываются в скелетов → едешь дальше сильнее
```

## 12. Открытые ветки и известные пробелы

- **⚠ Условие победы не подключено.** Формально есть `MatchGoal` (победа = золото ≥ **1000** И «пройти врата»), но он читает **экономику старого Camp** (`Camp.CAMP_GROUP`), которого в комнатном режиме нет → читает 0 и не видит новый `GoldBank`. В room-режиме цель матча **де-факто не финализирована** — нужно либо перецепить MatchGoal на GoldBank, либо задизайнить явную цель (дойти до врат / накопить N золота). Баннер «Золото для победы 0/1000» сейчас — легаси-cruft.
- **«Переплата золотом → рост репутации»** — слой к торговле; обсуждали, отложили.
- **Команды отряду** в room-режиме помимо эскорта (Hold/Defend) — каркас в `Squad` есть, UI-команд нет.
- **Полировка стола** — анимация падения монет, портрет гнома, дерево/пергамент.
- **Прочие добрые дела** как источники скидки (сейчас один — убийство гиганта).
- **Прочие заклинания** — разблокировка/прогрессия (сейчас только Искра).
- **Последняя металл-дверь (Room4→далее)** и контент за RoomWest — механика/наполнение не финализированы.
